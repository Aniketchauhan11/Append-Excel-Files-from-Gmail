# 📊 Append Excel Files from Gmail using Power Query

## 🚀 Overview

Built an automated solution to combine multiple Excel files received via Gmail into a single structured dataset using Power Query.

---

## ❗ Problem

Daily Excel reports received via email required manual downloading and consolidation, leading to:

* Time consumption
* High chances of errors
* Repetitive manual work

---

## 💡 Solution

Used Power Query to:

* Connect folder containing Gmail attachments
* Automatically detect and import all Excel files
* Clean and transform data
* Append into one master dataset
* Enable one-click refresh

---

## ⚙️ Tools Used

* Microsoft Excel
* Power Query

---

## 🔑 Key Features

* ✔ Automatic file consolidation
* ✔ Dynamic updates (new files auto-added)
* ✔ Data cleaning & transformation
* ✔ No manual intervention required
* ✔ Scalable solution

---

## 📂 Workflow

1. Download Gmail attachments into a folder
2. Connect folder with Power Query
3. Combine & transform data
4. Load final dataset
5. Refresh when new files arrive

---

## 📈 Output

A clean, combined dataset ready for analysis and reporting.
